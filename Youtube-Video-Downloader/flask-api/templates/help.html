This is an youtube downloader api that extracts video links for youtube videos

To use this api you need to paste the youtube video link after the data parameter

ex:- yt-downloader.com/data?link=https://www.youtube.com/watch?v=mISvJhV8w-w

and it will return the response in JSON format
